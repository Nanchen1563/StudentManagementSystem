日期	核心学习内容	每日学习要点
第1天	开发环境搭建 & 基础语法	安装JDK、配置环境变量、选择IDE（推荐IntelliJ IDEA或Eclipse），并学习变量、数据类型、运算符等核心基础。
第2天	面向对象编程 (OOP)	重点攻克类与对象、封装、继承、多态等核心概念，这是理解框架的基础。
第3天	核心API	学习字符串(String)、集合框架(List, Set, Map)、异常处理。其中集合框架尤为重要。
第4天	MySQL & SQL基础	学习MySQL的安装与使用，掌握数据库的增删改查（CRUD） 操作。
第5天	JDBC & 多表查询	学习用JDBC连接和操作数据库，并掌握多表联合查询（JOIN）技巧。
第6天	复习与综合练习	利用这一天巩固前5天的所有知识点，可以尝试写一个简单的控制台程序，如学生管理系统，来加深理解
